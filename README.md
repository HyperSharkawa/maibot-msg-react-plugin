# msg_react_plugin

MaiBot 的贴表情插件，让麦麦能对群聊消息贴表情。

## 安装步骤
1. 将仓库克隆/下载至麦麦的 `plugins` 目录下：
   ```powershell
   git clone https://github.com/HyperSharkawa/maibot-msg-react-plugin.git
   ```
2. 重启麦麦。

## 配置项说明
| 字段 | 类型 | 默认值 | 说明                  |
| --- | --- | --- |---------------------|
| `action_require` | str | 详见源码 | 影响主动 `msg_react` 动作的决策提示 |
